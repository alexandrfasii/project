Reference folder for reusable UI patterns used in this project.

|-------------------|-------------------|---------------------|

| Portfolio Cards   | styles/sections.css   | `.portfolio-card` |
| Process Steps     | styles/sections.css   | `.process-step` |
| Testimonial Cards | styles/sections.css   | `.testimonial-card` |

## Design tokens

All design tokens (colors, spacing, typography, shadows) are defined in `styles/main.css` under `:root`.

To change brand colors: update `--accent-1`, `--accent-2`, `--gradient`, `--gradient-text`.
